# Week 2 Grade Calculator

A student grade calculator program built for Week 2: Control Flow & Data Structures.

## Features

- Validates the number of students and subject marks
- Uses `if/elif/else` for grade calculation
- Stores student names, marks, and results in lists
- Uses loops to collect and display student data
- Calculates class statistics: average, highest, lowest
- Displays results in a formatted summary table

## Files

- `grade_calculator.py` - main program
- `test_students.txt` - sample input values
- `results_sample.txt` - example output
- `.gitignore2` - files and folders to ignore

## Usage

1. Open a terminal in the `week2-grade-calculator` folder.
2. Run the program with:

```bash
python grade_calculator.py
```

3. Follow the prompts to enter the number of students, student names, and marks.

## Sample Input

Use the contents of `test_students.txt` if you want to simulate the example.

## Notes

- Marks must be between 0 and 100.
- Student names cannot be empty.
