# Personal Information Manager

## 📌 Project Overview
This is my first Python project, designed to practice working with:
- Variables and data types  
- User input and validation  
- Basic calculations  
- Displaying formatted output  

The program stores static personal information, asks the user for their favorite food and color, validates the input, and then displays all the information neatly.

---

## 🛠 How to Run
1. Clone or download this repository.  
2. Navigate to the `personal_info` folder.  
3. Run the program using:
   ```bash
   python personal_info.py
